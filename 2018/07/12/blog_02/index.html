Hexo + Github + next 搭建博客（进阶）

发表于 2018-07-12

目的

把自己的blog变得好看，而且易于管理

学习资料

  1. Z皓博客
  2. next

环境配置

# 接着上一篇的操作 在blog目录下 右键点击Git Base Here
git clone https://github.com/iissnan/hexo-theme-next themes/next

实践操作

  1. 配置blog目录下的_config.yml

    theme: next        # 选择主题为next

title: Blog        # 设置Blog主题
subtitle:
description:
author: speakfly   # 设置作者名
language: zh-Hans  # 设置为中文
timezone:

  2. 配置blog/theme/next目录下的_config.yml

    #设置显示菜单
menu:
home: / || home
about: /about/ || user
tags: /tags/ || tags
#categories: /categories/ || th
archives: /archives/ || archive

  3. 生成显示标签的页面

    hexo new page tags # 会生成tags文件夹和index.html文件

# 修改 blog/source/tags/index.html

---
title: tags
date: 2018-07-12 14:29:25
type: "tags"
---

  4. 生成显示about的页面

    hexo new page about

# 修改 blog/source/about/index.html

---
title: about
date: 2018-07-12 14:31:33
---
#关于我
一个学习前端知识的小菜鸟

email: 1090384046@qq.com

  5. 生成静态文件并部署到远程仓库

    hexo g -d

  6. 在浏览器输入 https://用户名.github.io
